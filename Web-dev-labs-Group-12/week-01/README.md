# Week 01 - HTML Document Structure

## Objective
Reinforce HTML fundamentals by building a structured, multi-page website that demonstrates document structure, navigation, and multimedia usage.

## What Was Done
- Created the `week-01` folder structure for the lab submission.
- Added four HTML pages: `index.html`, `about.html`, `contact.html`, and `external.html`.
- Added proper HTML document structure with `<!DOCTYPE html>`, `html`, `head`, and `body` elements.
- Included title and meta tags on every page.
- Added headings, paragraphs, comments, and lists where required.
- Implemented internal navigation links across the four pages.
- Added external links to the required mall websites on the external page.
- Added one local image and one Unsplash image on the home page.
- Placed browser screenshots inside `assets/screenshots` for documentation.
